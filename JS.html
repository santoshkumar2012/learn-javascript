DEEP COPY AND SHALLOW COPY
Shallow Copy – Creates a new object but copies references to nested objects. Deep Copy – Creates a new object and recursively copies all values, ensuring no references remain. Difference - sh - References copied (not duplicated), deep - Entire structure duplicated sh - Changes in the copy affect the original, deep - Changes in the copy do NOT affect the original sh - Faster (less memory used) , deep - Slower (more memory used) -- Shallow copy examples let obj1 = { name: "John", details: { age: 25 } }; let obj2 = obj1; // Copies reference obj2.details.age = 30; console.log(obj1.details.age); // Output: 30 (Changes reflect in obj1) -- Deep copy let obj1 = { name: "John", details: { age: 25 } }; let obj2 = structuredClone(obj1); obj2.details.age = 30; console.log(obj1.details.age); // Output: 25 (Unaffected)

DUPLICATE NUMBER
function findDuplicates(arr) { arr.sort((a, b) => a - b); let duplicates = []; for (let i = 0; i < arr.length - 1; i++) { if (arr[i] === arr[i + 1] && !duplicates.includes(arr[i])) { duplicates.push(arr[i]); } } return duplicates; } // Example console.log(findDuplicates([1, 2, 3, 4, 2, 5, 6, 3, 8, 1])); // [1, 2, 3]
