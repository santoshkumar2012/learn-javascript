Best for overall page layouts, grids, and complex structures. example - .container { display: grid; grid-template-columns: repeat(3, 1fr); grid-template-rows: auto; gap: 10px; } // 1fr is fraction unit just like 200px width // Automatically adjusts based on available space. FlexBox A one-dimensional layout system (handles either rows or columns, not both at once). Best for aligning items in a row or column, great for components. display:flex ------------------------------------------------------- Media Queries - in CSS allow us to apply different styles based on the screen size, resolution, or device type.
